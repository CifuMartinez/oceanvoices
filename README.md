# Ocean Voices - Web Development

Sitio web del proyecto Ocean Voices, un short-film documental que da voz al mar y a las comunidades que dependen de él.

## Estructura del Proyecto

- `index.html` - Página principal del proyecto
- `cuaderno-bitacora.html` - Página del cuaderno de bitácora
- `equipo.html` - Página del equipo
- `css/` - Estilos CSS
- `js/` - Scripts JavaScript
- `images/` - Imágenes y recursos visuales
- `fonts/` - Fuentes personalizadas (Chivo Mono)
- `assets/` - Assets SVG e iconos
- `videos/` - Videos del proyecto
- `gsap/` - Librerías GSAP para animaciones

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- GSAP (GreenSock Animation Platform)
- Swiper.js (Para carruseles)

## Características

- Diseño responsive
- Animaciones con GSAP
- Sliders interactivos
- Navegación entre páginas

## Desarrollo

Abre los archivos HTML directamente en el navegador o utiliza un servidor local para desarrollo.
